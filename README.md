# 🎙️ Pronunzia Master — English Pronunciation Guide

Aplicación web bilingüe (Español/Inglés) e interactiva para aprender y practicar las reglas de pronunciación fonética del inglés. Incluye síntesis de voz nativa, reconocimiento de voz para evaluar tu pronunciación en tiempo real, y una base de datos completa de reglas fonéticas.

## ✨ Características

- **100% bilingüe**: Cambia entre Español e Inglés con un solo clic.
- **Más de 100 reglas fonéticas**: Vocales cortas y largas, diptongos, consonantes clave, letras mudas, acentuación, entonación y habla conectada.
- **Escucha nativa (Text-to-Speech)**: Usa la Web Speech API para reproducir la pronunciación correcta de cada palabra de ejemplo.
- **Práctica de pronunciación (Speech Recognition)**: Graba tu voz con el micrófono y recibe retroalimentación instantánea sobre si tu pronunciación coincide con la palabra objetivo.
- **Buscador y filtros por categoría**: Encuentra reglas específicas por palabra clave, símbolo fonético o categoría.
- **Galería de imágenes ampliable**: Haz clic en la imagen de cada regla para verla en pantalla completa.
- **Diseño responsivo**: Funciona en escritorio, tablet y móvil.

## 🗂️ Categorías incluidas

- Vocales Cortas
- Vocales Largas
- Diptongos
- Consonantes Clave
- Letras Mudas
- Fonética Avanzada (acentuación, entonación, sufijos, habla conectada, etc.)

## 🛠️ Tecnologías utilizadas

- **HTML5 / CSS3**
- **Tailwind CSS** (vía CDN)
- **JavaScript (Vanilla)**
- **Web Speech API** — síntesis y reconocimiento de voz (requiere navegador compatible, como Chrome o Edge)
- **Google Fonts (Inter)**

## 🚀 Cómo usarlo

1. Clona o descarga este repositorio.
2. Abre el archivo `index.html` en tu navegador (recomendado: Google Chrome o Microsoft Edge, por soporte completo de reconocimiento de voz).
3. Explora las reglas fonéticas, escucha los ejemplos y practica con el micrófono.

> ⚠️ El reconocimiento de voz requiere permisos de micrófono y conexión a internet (usa los servicios de reconocimiento del navegador).

## 📋 Requisitos

- Navegador moderno con soporte para `SpeechSynthesis` y `SpeechRecognition` (Chrome/Edge recomendados).
- Conexión a internet (para cargar Tailwind CSS, Google Fonts, y el motor de reconocimiento de voz).

## 📌 Notas de desarrollo

Este proyecto usa Tailwind CSS vía CDN, lo cual es ideal para prototipos y demos, pero **no se recomienda para producción**. Para un entorno de producción, se sugiere instalar Tailwind como plugin de PostCSS o usar Tailwind CLI. Más información: [https://tailwindcss.com/docs/installation](https://tailwindcss.com/docs/installation)
